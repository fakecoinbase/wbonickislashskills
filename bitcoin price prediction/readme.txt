__The main goal of this project is to build a reliable model that predicts the price of bitcoin.__

data_cleaning - a folder where the raw data downloaded from the Internet are stored, is contains a jupyternotebook file, in which data are cleaned and the dataset is created (all prices with datetime index)
BTC_data.CSV - the main dataset with prices and dates
BTC_multilinear_regression - a notebook with multilinear regression bitcoin price model
BTC_nonlinear-regression - an extension of BTC_multilinear_regression notebook where nonlinear term are introduced
